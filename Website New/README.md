# 🚗 AutoBill Pro — Garage Billing & Inventory Management

Premium futuristic React.js website for automotive workshop management.

## Tech Stack
- React 19 + Vite 8
- Three.js + @react-three/fiber + @react-three/drei (3D Car)
- Framer Motion (animations)
- GSAP (scroll effects)
- Tailwind CSS v4
- Lenis (smooth scroll)
- Lucide React (icons)

## Sections Built
1. Hero — 3D Interactive Sports Car + Stats
2. Trust/Stats — 10K+ Bills, 500+ Workshops, 99% Accuracy
3. About — Software explanation with metrics
4. Features — 6 glassmorphism cards (Dashboard, Inventory, Billing, Expenses, Reports, CRM)
5. Dashboard Preview — Futuristic live UI mockup with charts
6. How It Works — 6-step animated timeline
7. Analytics — Charts, KPIs, Top Services, Mechanic Rankings
8. Future Features — WhatsApp, Payments, Mobile App, Mechanic Tracking
9. Testimonials — 3 premium glass cards
10. Contact — Glowing form with glassmorphism
11. Footer — Multi-column professional footer

## Run Locally
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Build for Production
```bash
npm run build
npm run preview
```
